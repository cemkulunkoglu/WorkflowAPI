# WorkflowAPI

WorkflowAPI; akış (workflow) tasarımı, akış node/edge yönetimi ve ilgili süreç/izin/rol gibi işlevleri sağlayan .NET Web API servisidir.

## Teknolojiler
- .NET (Web API)
- Entity Framework Core
- MySQL
- RabbitMQ / SignalR / JWT

## Gereksinimler
- .NET SDK 8
- MySQL Server
- Docker Desktop
